# Fitness Trainer Landing (Next.js Starter)

Minimal starter for a modern, dark, premium-style landing page built with Next.js App Router, TypeScript, and SCSS modules.

## Run locally

```bash
npm install
npm run dev
```

## Structure

- `app/` - App Router pages and layout
- `components/` - Landing page sections + reusable UI
- `styles/` - Global styles and SCSS variables
- `public/` - Static assets
- `utils/` - Shared site configuration
